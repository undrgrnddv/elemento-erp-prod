Cómo Instalar un Modulo Usando GitHub
-----------------------------------------------------------------------------------------
1. Obtén la URL del Repositorio ej. https://github.com/OCA/reporting-engine.git
2. Navega al Directorio de Addons de tu Proyecto
3. Clona el Repositorio
    La sintaxis general es: 
        git clone -b <nombre_de_la_rama> <url_del_repositorio> <nombre_de_la_carpeta_de_destino>
    Ejemplo:
        git clone -b 18.0 https://github.com/OCA/reporting-engine.git oca-reporting-engine
4. Agrega la ruta al directorio del repositorio en tu archivo de configuración de Odoo addons_path (odoo.conf).
5. Reiniciar el Servicio/Servidor de Odoo.
6. Actualizar la Lista de Módulos de Odoo, con el Modo Desarrollador activado. 
9. Busca los módulos que quieras instalar en el menú de aplicaciones y haz clic en Instalar.
